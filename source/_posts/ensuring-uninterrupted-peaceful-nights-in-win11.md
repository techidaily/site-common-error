## Run Interrupted: Script Halted

![the code execution cannot proceed](https://images.drivereasy.com/wp-content/uploads/2021/11/the-code-execution-cannot-proceed.jpg)

 The code execution cannot proceed error usually occurs because a**specific DLL file was not found** . It can be for a range of DLL files such as MSVCP140.dll, MSVCP120.dll, VCRUNTIME140.dll and so on. If you encounter this issue when launching or closing a program, don’t worry. After reading this post, you’ll know how to fix it easily and quickly.

## Try these fixes

 Here are 4 simple fixes for the code execution cannot proceed error message no matter which DLL files is missing. You don’t have to try them all. Just work your way down the list until you find the one that solves the problem.

1. **[Reinstall the program](#fix1)**
2. **[Repair your system files](#fix2)**
3. **[Install the DLL file manually](#fix3)**
4. **[Update your device drivers](#fix4)**

## Fix 1 – Reinstall the program

 When you receive the code execution cannot proceed error alert, it usually suggests you reinstall the program to fix the issue. You can simply reinstall the program you’re having a problem with to see if that’s the cause. To make sure you perform a clean reinstallation, follow the steps below:

1. On your keyboard, press the**Windows logo key** and**R** at the same time to invoke the Run command. Then type**appwiz.cpl** and click**OK** .  
![](https://images.drivereasy.com/wp-content/uploads/2020/12/6-1-1.jpg)
2. Right-click the program and click**Uninstall** .  
![](https://images.drivereasy.com/wp-content/uploads/2020/12/6-2-1.jpg)
3. On your keyboard, press the **Windows logo key** and **R** at the same time to invoke the Run command. Type**%programdata%** in the field and press**Enter** .  
![](https://images.drivereasy.com/wp-content/uploads/2021/11/appprogramdata.jpg)
4. Locate the program folder and delete it.
5. Again, press the **Windows logo key** and **R** on your keyboard to open the Run box. Type**%appdata%** and press**Enter** .  
![](https://images.drivereasy.com/wp-content/uploads/2021/11/appdata.jpg)
6. Locate the program folder and delete it.
7. Restart your computer for the changes to take effect.
8. Go to the official source of the program. Then download and reinstall it manually.

 Launch the program to test for the issue. If it reoccurs, move on to the second method.

### Option 2 – Manually

 Below we’ll show you how to run the System File Checker and DISM.exe tool step by step.

1. Click the**Start** button. Type**cmd** in the Windows search bar. Then right-click Command Prompt and select**Run as administrator** .  
![](https://images.drivereasy.com/wp-content/uploads/2020/07/10-1.jpg)
2. Click**Yes** when you’re prompted to continue.  
![](https://images.drivereasy.com/wp-content/uploads/2020/07/5-2-4.jpg)
3. Type the following command and press**Enter** .  
sfc /scannow
4. Wait a few minutes for the process to complete. Then you may see a message that says **Windows Resources Protection found corrupt files and successfully repaired them** .
5. Launch the problematic application to see if it works. If not, you can continue to troubleshoot with the DISM tool.
6. Run**Command Prompt as administrator** again and enter the commands as below:  
dism.exe /online /cleanup-image /scanhealth  
 This will scan the Windows component store corruption of your PC.  
dism.exe /online /cleanup-image /restorehealth  
 This will restore the health of your PC.

 Now you should be able to get everything fine with the system files and the error code should be gone. If not, check out the fix below.

## Fix 3 – Install the DLL file manually

 If recovering the DLL file doesn’t solve the problem, you may try to install the DLL file manually. To find the correct and safe DLL files, simply check out the DLL-files.com.

1. Go to the **[DLL-files.com](https://tools.techidaily.com/drivereasy/download/)**  .
2. Type the specific DLL file that causes the error in the search bar (MSVCP140.dll for example), and click**Search DLL file** .  
![](https://images.drivereasy.com/wp-content/uploads/2021/11/dll-file-1.jpg)
3. Select the latest DLL file that’s corresponding with your operating system and click**Download** .  
![](https://images.drivereasy.com/wp-content/uploads/2021/11/dll-file-2-1.jpg)
4. After it’s downloaded, extract the zip file to any location you like on your computer. Then copy the DLL file from the folder and paste it to**C:\\Windows\\System32** as well as the**install folder of the game or application** that requests the file.
